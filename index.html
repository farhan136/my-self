Saya adalah seorang Backend Developer dengan pengalaman lebih dari tiga tahun dalam pengembangan aplikasi web dan API. Selama perjalanan karier saya, saya telah terlibat dalam berbagai proyek berskala besar seperti HRIS, sales order system, aplikasi IT service management, hingga integrasi sistem CCTV untuk pemerintahan. Keahlian utama saya mencakup PHP (Laravel, CodeIgniter, Symfony), Node.js, serta pengelolaan database MySQL dan PostgreSQL. Saya terbiasa bekerja lintas departemen dan berkolaborasi dengan tim DevOps untuk proses deployment yang efisien.

Saya dikenal cepat beradaptasi dengan teknologi baru dan memiliki logika pemrograman yang kuat. Saya juga terbuka terhadap masukan dan senang bekerja dalam tim yang dinamis untuk menciptakan lingkungan kerja yang positif dan produktif.
